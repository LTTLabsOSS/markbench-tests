# Browser Benchmark

Runs one of several browser based benchmarks in Google Chrome

## Prerequisites

- Python 3.10+
- Google Chrome

## Options

- `--benchmark`: One of 6 options of benchmarks [jetstream2,speedometer,motionmark,kraken,webxprt4,webxprt5]

## Output

report.json - an array of json objects, each being a report per scene benchmarked.
